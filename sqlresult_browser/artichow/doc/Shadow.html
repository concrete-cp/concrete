  • public draw(Driver $driver, Point $p1, Point $p2, int $mode)
    Dessine l'ombre avec le pilote $driver dans un rectangle dont la diagonale est le segment qui relie les points $p1 et $p2. Le paramètre $mode détermine le mode d'affichage de l'ombre. Si Shadow::OUT est spécifié, alors l'ombre sera dessinée en dehors du rectangle. Si Shadow::IN est spécifié, alors l'ombre sera dessinée à l'intérieur du rectangle.
